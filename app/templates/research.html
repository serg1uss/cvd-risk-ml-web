Research results

This page summarizes the experimental findings for multiclass CVD risk prediction. The model assigns three class labels: HIGH, INTERMEDIARY (Medium), and LOW.

Dataset overview

The study frames CVD risk as a three-class classification problem. The quick test uses six user inputs and a derived BMI value, aligning a compact questionnaire with the core predictive signals used by the full model.

Class distribution

The class distribution shows imbalance across the three labels; this motivates class balancing strategies and the use of macro-averaged metrics to avoid dominance by the majority class.

Exploratory data analysis (EDA)

The correlation heatmap highlights expected relationships such as BMI with weight and height. Correlated variables are common in tabular medical data and should be interpreted carefully during modeling.

Model performance

The confusion matrix indicates typical errors between adjacent risk categories, particularly INTERMEDIARY versus HIGH or LOW, which is expected in a three-class clinical risk setting.

Key metrics
  • Accuracy: 0.55
  • Macro-F1: 0.49
  • Weighted-F1: 0.56

Feature importance

The top-ranked features explain most of the model signal, supporting the quick-test design that focuses on a compact set of high-impact inputs.

Limitations

  • Educational demonstration only; not a diagnostic tool.
  • Quick test uses a limited feature set compared to full clinical assessments.
  • Performance depends on data quality and representativeness.
  • Class imbalance can affect minority-class recall.

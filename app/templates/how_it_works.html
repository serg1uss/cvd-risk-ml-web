How it works

  1. User inputs 6 parameters (Age, Height, Weight, Activity, Smoking, Family history). BMI is derived from height & weight if the model expects it.
  2. Server validates numeric ranges and categorical options.
  3. CatBoost model generates a risk class (and probabilities if available).
  4. Website displays the result and interpretation with disclaimers.
